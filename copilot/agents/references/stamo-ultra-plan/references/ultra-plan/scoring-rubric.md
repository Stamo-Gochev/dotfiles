# Ultra Plan Scoring Rubric

Score each criterion as a percentage (0-100), then apply weights for weighted totals (%).

## Criteria and Weights

1. Requirement Coverage (weight 0.25)
   Does the plan map to all stated requirements and constraints?
2. Feasibility and Sequencing (weight 0.20)
   Are dependencies explicit and phase order realistic?
3. Risk Quality (weight 0.20)
   Are risks concrete, prioritized, and mitigated with owners/triggers?
4. Verification Strength (weight 0.15)
   Are validation steps measurable and phase-aligned?
5. Assumption Discipline (weight 0.10)
   Are assumptions explicit, testable, and minimized?
6. Contingency and Rollback (weight 0.10)
   Is there a practical fallback path if assumptions fail?

## Output Format

- Total percentage per planner
- Weighted total percentage per planner
- Criterion-level notes per planner
- Confidence level in final synthesis (Low/Medium/High)
- Tie-break rationale when top 2 weighted scores are within 5 percentage points
